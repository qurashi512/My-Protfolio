# 🚀 Gorashe Suliman - Portfolio Website

Welcome to my professional portfolio! This is a fully responsive, multilingual web application showcasing my skills, projects, and experience in IT and programming.

![Portfolio Preview](images/profile-photo.jpg)

## 🌟 Features

### 🎨 Design & UX
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional, and visually appealing interface
- **Smooth Animations**: Subtle animations and transitions for better UX
- **Accessibility**: WCAG 2.1 compliant with semantic HTML
- **Profile Photo**: Professional circular profile picture with hover effects

### 🌍 Multilingual Support
- **English (EN)** - Default language
- **German (DE)** - Full German translation
- **Language Persistence**: Selected language saved in localStorage
- **Real-time Switching**: Instant language switching without page reload

### 📱 Responsive Components
- **Mockup Displays**: Professional desktop and mobile mockups for projects
- **Mobile-First Approach**: Built with mobile-first responsive design
- **Flexible Grid Layouts**: Using CSS Grid for optimal layouts
- **Profile Photo Support**: Optimized for all screen sizes

### 🔍 Interactive Features
- **Contact Form**: Integrated mailto functionality
- **Social Media Links**: Direct links to all social platforms
- **Smooth Scrolling**: Enhanced navigation experience
- **Scroll to Top Button**: Quick navigation for long pages
- **Hamburger Menu**: Mobile-friendly navigation menu
